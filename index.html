Link Type Examples

Icon font created from IcoMoon by KISS Web Design.

There are 3 main ways of using the icons:

  • Web Font
  • SVG Images
  • PNG Images

Additionally, the icon font can be displayed according to

  • File type
  • Link type (only applies to mailto:)
  • Class name

The SVG and PNG images are 32px x 32px, and are avaiable as sprites or individually. Sprites have been used here.

If class names are required, they are shown next to the icon.

For users with IE7 or lower there is an extra stylesheet and javascript file to be included - use conditional comments to only include them for the browsers than need them
e.g.
<!--[if lt IE 8]>
<link rel="stylesheet" href="stylesheets/ie7.css">
<!--<![endif]-->
