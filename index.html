Link Type Examples

Icon font created from IcoMoon by KISS Web Design.

There are 3 main ways of using the icons:

  • Web Font
  • SVG Images
  • PNG Images

Additionally, the icon font can be displayed according to

  • File type
  • Link type (only applies to mailto:)
  • Class name

The SVG and PNG images are available in 16px, 32px, 48px and 64px sizes, and are avaiable as sprites or individually. Sprites have been used here.

If class names are required, they are shown next to the icon.

For users with IE7 or lower there is an extra stylesheet and javascript file to be included - use conditional comments to only include them for the browsers than need them
e.g.
<!--[if lt IE 8]>
<link rel="stylesheet" href="stylesheets/ie7.css">
<!--<![endif]-->

By class name

The class naming convention is "icon-type(optional size)-name" eg icon-font-pdf is the class name for using the web font for a pdf (note: web fonts do not require a size). If you wanted a 32px svg image of the pdf icon you would use icon-svg32-pdf. Simples!
