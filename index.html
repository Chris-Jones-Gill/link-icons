Link Type Examples

This project has been created by KISS Web Design, and includes icons taken from the IcoMoon Icon Font and the FontAwesome Icon Font.

There are 3 main ways of using the icons:

  • Web Font (svg, woff, ttf, eot - in that order of preference)
  • SVG Images (in case you dont want to use a font)
  • PNG Images (only use as a final fallback)

Web Font should be your first choice, as they can be styled in the same way as any text. eg. color, size, weight, text-shadow, etc. Whereas the image files are only supplied in black.

Additionally, the icon font can be displayed according to

  • File type
  • Link type (only applies to mailto:)
  • Class name

File and Link type are only applicable to the Web Font, and are applied to any $lt;a href... link. If you don't want to use the icon you can add the class "noIcon".

This link is for a PDF there is no icon because the "noIcon" class has been used.

The SVG and PNG images are available in black only and at 16px, 32px, 48px and 64px sizes, they are avaiable as sprites or individually. Sprites have been used here.

If class names are required, they are shown next to the icon.

For users with IE7 or lower there is an extra stylesheet and javascript file to be included - use conditional comments to only include them for the browsers than need them
e.g.
<!--[if lt IE 8]>
<link rel="stylesheet" href="stylesheets/ie7.css">
<!--<![endif]-->

By class name

The class naming convention is "kwd-icon-type(optional size)-name" eg kwd-icon-font-pdf is the class name for using the web font for a pdf (note: web fonts do not require a size). If you wanted a 32px svg image of the pdf icon you would use kwd-icon-svg32-pdf. Simples!
